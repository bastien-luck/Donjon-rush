Donjon rush

Plateforme utilisé :
Windows 7

Logiciel de programmation :
codeblocks

Ce projet a été réalisé lors de mon année d'intégration à l'ENSIBS (année de remise à niveau).

Pour toute information sur le jeu, se référencer à la synthèse du projet.
Les règles du jeu sont aussi expliqué dans la synthèse (hors stats des équipements).
Le jeu se joue en grande parti avec le pavé numérique (les autres input disponible sont visibles à chaque affichage).
Le jeu possède un tutoriel pour connaitre les bases du jeu sans avoir à lire toute la synthèse :).

Aucune documentation a été faite à l'époque car je ne connaissais pas le principe et le projet ne prenait pas cela en compte dans sa notation.

Le jeu n'est pas vraiment fini mais ne sera pas modifié donc il est considéré en version finale.

Ce projet a été fait par rapport à un cours d'où son orientation console et gameplay.
Certains fichiers du rendu ne sont pas présent car je tiens à limiter aux sources, exécutable et rapport dans mes projets git hub.
